# ChatGPT with Slack

ChatGPT에서는 아직 한국어 자료 데이터가 부족해서 질문할 때마다 부정확한 답변이 달린다.<br>
보이저엑스 남세동 대표의 말로는 한국어 자료는 1.5%가 들어갔다고 언급도 했다. <br>
